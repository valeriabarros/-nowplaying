# #nowplaying
Single page application that allows to see and post #nowplaying tweets.


## How to

**First of all, make sure your port 3000 is available!**

```git clone git@github.com:valeriabarros/nowplaying.git```

```cd nowplaying```

```npm install```

```npm run serve```

Take a look in your chrome!
